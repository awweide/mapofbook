# Map of Book

Static GitHub Pages visualization for chapter-level character importance in `lotr.txt`.

## Local preview

Open `index.html` directly in your browser, or run a simple static server:

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000>.

## GitHub Pages setup

1. Push this repository to GitHub.
2. In repository **Settings → Pages**.
3. Set **Source** to **Deploy from a branch**.
4. Choose branch (usually `main`) and folder `/ (root)`.
5. Save and wait for deploy.

Your site will be available at:

`https://<your-username>.github.io/<repo-name>/`
